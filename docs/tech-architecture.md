# Tech Architecture

## ESP Recommendation: Brevo (Sendinblue)

**Selected** after comparing 5 providers against 6 criteria relevant to our 60-client model.

### Comparison Summary

| ESP | Pricing (60 clients) | Sub-accounts | API | Templates | SMS |
|-----|---------------------|--------------|-----|-----------|-----|
| **Brevo** вӯҗ | **~$90/mo** ($1.50/client) | вң… Enterprise | вӯҗвӯҗвӯҗвӯҗвӯҗ | вӯҗвӯҗвӯҗвӯҗ | вң… Built-in |
| SendGrid | ~$90/mo ($1.50/client) | вң… Sub-users | вӯҗвӯҗвӯҗвӯҗвӯҗ | вӯҗвӯҗвӯҗ | вқҢ Separate |
| MailerLite | ~$159/mo ($2.65/client) | вң… Advanced | вӯҗвӯҗвӯҗвӯҗ | вӯҗвӯҗвӯҗвӯҗвӯҗ | вқҢ None |
| Mailchimp | ~$350+/mo ($5.83+/client) | вң… Premium | вӯҗвӯҗвӯҗвӯҗ | вӯҗвӯҗвӯҗвӯҗвӯҗ | вӯҗвӯҗвӯҗ |
| AWS SES | ~$12/mo ($0.20/client) | вқҢ Custom build | вӯҗвӯҗвӯҗ | вқҢ None | вқҢ SNS |

### Why Brevo
- **Per-email pricing** (not per-contact) вҖ” best value as client lists grow
- **Sub-accounts** via Enterprise plan for per-client isolation
- **Built-in SMS** for our $200/mo upsell
- **REST API** with Node.js SDK for full automation
- **Good deliverability** with dedicated IP options

| Scheduler | node-cron |
